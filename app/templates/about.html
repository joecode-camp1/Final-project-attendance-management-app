About This Project

The Student Attendance Management System (AttendanceMS) is a web application designed to simplify and improve how student attendance is recorded and managed in educational institutions.

🎯

Project Purpose

This system was developed to reduce paper-based attendance tracking, minimize human error and provide a more efficient digital solution.

👨‍💻

Developers

This project was built by Joseph Acquah and Christian Avorti as part of their CodeCamp tuition training.

🚩

Our Goal

To make attendance tracking more accurate, faster and easier while improving reliability.
